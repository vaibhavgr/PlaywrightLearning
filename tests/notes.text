Child window concept : 
In Playwright, there is NO concept called child window.
Playwright only knows PAGES.
One browser
One tab = one page
New tab opens = new page
So Playwright must be told:
👉 “Hey, I am about to open a new tab — please catch it.”
The ONLY pattern you need to remember ⭐
WAIT for new page + CLICK at the same time
We do this using Promise.all.

Promise (microtask) > setTimeout (macrotask)
console.log("A");
setTimeout(() => console.log("B"), 0);
Promise.resolve().then(() => console.log("C"));
console.log("D");

output : A D C B
 Queues me kya gaya?

setTimeout → Callback Queue (macrotask)

Promise.then → Microtask Queue 🔥 (high priority)

Event loop ka rule:

👉 Call stack empty hote hi:

Microtask queue execute hoti hai
👉 "Promise VIP hai, setTimeout general line me hai" 😄
